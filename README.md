# Lipify--JupyterNotebooks
A Lip-Reading Application to help interpret videos based on GRID Corpus Dataset, The project uses visual cues as opposed to most audio-visual cues present in most datasets


### Dataset Info:

We use the GRID Corpus dataset which is publicly available at this link 
You can download the dataset using our script: `Download_Grid_Corpus.sh` 

To Download please run the following line of code in your terminal: 
```
bash Download_Grid_Corpus.sh FirstSpeaker SecondSpeaker 
```
where FirstSpeaker and SecondSpeaker are integers for the number of speakers to download
